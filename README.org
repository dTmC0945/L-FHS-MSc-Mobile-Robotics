* Lecture Information

- The goal of this lecture is to give you a better view of how mobile robotics works, the
  different challenges it faces and methods to overcome them.
- This lecture you will use ROS, a versatile software capable of doing pathfinding and
  localisation, a perfect fit for use in mobile robotics.

* Requirements and the Learning Outcomes

- The student should be comfortable with working with either Python
  and/or C++ and should have an beginner level knowledge of the Linux
  environment and the command line interface
